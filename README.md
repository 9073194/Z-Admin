# Z-Admin

本项目基于 Layui, 适合需要快速开发的后端人员使用.

基础框架: [Layui](https://www.layui.com/)

配色参考: [Vue Element Admin](https://panjiachen.github.io/vue-element-admin/#/dashboard)

## 特色功能

* 动态面包屑
* `tab` 卡记忆
* 点击 `tab` 页, 选中左侧相应导航
* `tab` 页右键菜单
